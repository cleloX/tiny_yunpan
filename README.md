简易云盘：

技术栈：前后端分离
前端：vue2.0 + elementUI + axios发送网络请求
后端：spring框架

实现功能：
用户：文件列表，登录，文件上传、下载、全选、删除，分享（）；回收站，恢复，清空
管理员：添加，批量导入，文件列表，禁用
