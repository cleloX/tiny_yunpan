前端代码
左侧导航栏：1.点击根目录可进入云盘主页，2.点击下方的垃圾桶图标可以进入回收站
主页右上角“人头”图标：管理员可以进入用户管理界面（普通用户不显示这个功能）
管理员界面：用户添加，批量导入，用户状态管理。删除用户，用户请求管理
云盘主页：上传、下载、分享、删除、全选，单击文件选择，双击预览
回收站：恢复文件、清空回收站
